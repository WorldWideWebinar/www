## 💡 프로젝트 소개

#### 🛠 핵심 기능

* WebRTC(Openvidu)를 활용한 화상 회의 서비스

* 그래픽 서버에 탑재한 AI 모듈을 활용한 STT 자막/번역 서비스

* 각 팀별 메신저 기능 서비스

* AI를 활용한 요약 회의록 제공 서비스

#### 📢 기능 시연

![webrtc 시연](./WebRTC시연.jpg)
* 화면 공유, 각자의 음성/화상 공유 가능한 회의

![채팅 시연](./채팅기능.PNG)
* 팀 선택 후, 팀원간 대화 가능한 메신저

![회의록](./회의록.PNG)
* AI 기반으로 회의 전문을 요약해 제공하는 회의록
